User Interaction Descriptors

Manage how you interact with 
@contextdogs
View my content

This feature will soon not exist, as for a user to not view your content, you will need to change to a "private" account.

Interact with my content

Likes, comments, and reposts are the forms of interaction with a user's content.

Direct message me

Can send and receive direct messages from me. Enabling this for the first time will notify the other user to enable direct messaging as well.

Disappear from my feed

When you no longer want to view a user's content, but otherwise wish to leave the relationship unaffected.
